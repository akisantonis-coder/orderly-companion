# PWA Icons

Για να λειτουργήσει σωστά το PWA, χρειάζεσαι τα παρακάτω icons:

- `pwa-192x192.png` - 192x192 pixels
- `pwa-512x512.png` - 512x512 pixels

Μπορείς να χρησιμοποιήσεις οποιοδήποτε icon generator ή να δημιουργήσεις τα icons σου.

**Συνιστώμενες εργαλεία:**
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Το `pwa-512x512.png` θα χρησιμοποιηθεί και ως maskable icon για Android.
